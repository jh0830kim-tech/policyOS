# PolicyOS

PolicyOS is an AI Office operating system for elected officials, councils, public institutions, and policy teams.

## Current status
- Sprint 1: project foundation completed
- Sprint 2: authentication and identity completed
- Current backend: Python 3.12, FastAPI, SQLAlchemy Async, Alembic, PostgreSQL, Redis, Pytest, Ruff

## Core product areas
1. Identity and access management
2. Policy research and legislative work
3. Document and knowledge management
4. AI agent collaboration
5. Audit, security, and public-sector governance
