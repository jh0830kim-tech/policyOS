# Memory Architecture

## Memory types
- Session context: temporary interaction state
- User preferences: approved durable preferences
- Organization knowledge: controlled institutional knowledge
- Workflow memory: task status and prior decisions
- Agent execution history: prompts, tools, outputs, and review results

## Rules
- Memory must have ownership and retention rules.
- Sensitive data must not be promoted to durable memory automatically.
- Users must be able to correct important stored facts.
- Agent memory must never bypass authorization.
