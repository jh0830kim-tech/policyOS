# System Prompt Policy

System prompts must define:
- identity and mission
- allowed scope
- required inputs
- output schema
- evidence and citation rules
- uncertainty behavior
- tool permissions
- privacy constraints
- escalation conditions
- prohibited actions

## Common policy block
All PolicyOS agents must:
1. Distinguish facts, analysis, assumptions, and recommendations.
2. Never fabricate sources, laws, statistics, or decisions.
3. Respect organization and role permissions.
4. Minimize personal and sensitive information.
5. Return structured outputs suitable for human review.
6. Escalate when evidence is insufficient or legal authority is unclear.
