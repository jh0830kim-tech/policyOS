# Core Values

## Trust
Outputs must be explainable, source-aware, and reviewable.

## Service
Technology exists to improve public service outcomes.

## Security
Privacy, least privilege, and auditability are foundational.

## Clarity
Readable code and understandable documents are preferred over clever complexity.

## Continuity
PolicyOS preserves knowledge beyond individual staff changes.

## Learning
The system should support gradual improvement through tests, reviews, and measured feedback.
